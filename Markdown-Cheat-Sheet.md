## Basis Syntax 
### Heading
```javascript
# H1

## H2

### H3
```

### Bold
```javascript
**boldText**
```

### Italic
```javascript
*italicized text*
```

### Blockqoute 
```javascript
> blockquote
```

### Ordered List 
```javascript
1. First item
2. Second item
3. Third item
```

### Unordered List 
```javascript
- First item
- Second item
- Third item
```

### Code 
```javascript
```code```
```

### Horizontal Rule 
```javascript
---
```

### Link 
```javascript
[title](https://www.example.com)
```

### Image 
```javascript
![alt text](image.jpg)
```
